因缺少 API key 等原因不可用时，使用 DuckDuckGo Lite 页面配合 `web_fetch` 执行网页搜索。\n\n本 skill 迁移自上游仓库：`openclaw/skills -> skills/jakelin/ddg-web-search`，已按当前 G3KU 本地 skill 结构接入。\n\n## 何时使用\n- 需要联网搜索，但没有可用搜索 API key\n- `web_search` 失败，特别是提示缺少 Brave API key 时\n- 只需要普通网页检索结果，而不是图片、视频或复杂垂类搜索\n\n## 核心做法\n让代理直接访问 DuckDuckGo Lite 搜索页：\n\n- 基本查询：`https://lite.duckduckgo.com/lite/?q=<URL编码后的查询词>`\n- 可选语言：`&kl=us-en`、`&kl=cn-zh`\n- 分页：`&s=30`、`&s=60`\n\n优先使用 `web_fetch` 抓取结果页，再从页面中提取标题、跳转链接和摘要。\n\n## 推荐提示词模板\n可直接让模型按如下方式执行：\n\n1. 将查询词进行 URL 编码\n2. 构造 DuckDuckGo Lite URL\n3. 使用 `web_fetch` 获取页面\n4. 提取前 5–10 条结果\n5. 返回标题、链接、摘要，并给出简要结论\n\n示例：\n\n- 搜索：`site:docs.github.com GitHub Actions matrix strategy`\n- URL：`https://lite.duckduckgo.com/lite/?q=site%3Adocs.github.com%20GitHub%20Actions%20matrix%20strategy`\n\n## 使用说明\n当你需要搜索时，可显式说明：\n\n- “如果 `web_search` 不可用，请改用 DuckDuckGo Lite + `web_fetch` 搜索。”\n- “请使用 ddg-web-search 技能搜索以下内容：<query>”\n\n## 局限性\n- DuckDuckGo Lite 不稳定支持时间过滤\n- 主要是文本网页结果，不适合图片/视频搜索\n- 结果来源与排序可能和其他搜索引擎不同\n- 如果当前运行环境本身没有 `web_fetch` 或外网访问能力，此 skill 不能独立生效\n\n##
